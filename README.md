# FSR Signal Chain — 柔性压力传感信号链

基于 STM32F103C8T6 + HAL 库的柔性薄膜压力传感器（FSR）信号采集系统。大一暑假第二个嵌入式项目，面向柔性传感实验室电路组申请。

## 项目定位

用现成的 FSR 柔性压阻传感器，从零搭建完整的模拟信号采集链路：**分压 → 滤波 → 运放缓冲 → ADC 采集 → 软件标定 → OLED 显示**。

不是买模块连连看——每一步的信号链设计（分压电阻怎么选、为什么加运放缓冲、加与不加差多少）都有实测对比数据。

## 两步走方案

| | 第 1 步 | 第 2 步 |
|---|---|---|
| 电路 | FSR + 100kΩ 分压 → 直连 ADC | + RC 低通（~30Hz）+ MCP6002 跟随器 |
| 核心问题 | ADC 能读到压力数据吗 | 加运放缓冲后数据质量改善多少 |
| 学到什么 | ADC 采集、传感器标定、分段线性化 | 运放跟随器、阻抗匹配、信号链对比验证 |

## 硬件

| 元件 | 型号 |
|---|---|
| 主控 | STM32F103C8T6（ARM Cortex-M3） |
| 传感器 | RP-C7.6-ST 薄膜压力传感器（FSR）×2 |
| 运放 | MCP6002 轨到轨双运放（DIP-8，电压跟随器） |
| 显示 | 0.96 寸 OLED SSD1306（I2C, 0x3C） |
| 开发工具 | Keil5 + CubeMX + HAL 库 |

## 信号链路

```
FSR 受压 → 分压网络 (100kΩ) → RC 低通 (~30Hz) → MCP6002 电压跟随器 → STM32 ADC (12bit) → 软件滤波+标定 → OLED
                                                                                  │
                                                                           USART1 上传 PC
```

### 为什么需要运放缓冲

FSR 空载阻抗 ~MΩ 级。STM32 ADC 内部采样保持电容约 14pF，若信号源阻抗过高，采样电容建立不充分，读数偏移且噪声大。MCP6002 跟随器：输入阻抗极高（不抽信号电流），输出阻抗极低（轻松驱动 ADC 采样电容），增益 = 1，作用是**阻抗匹配而非放大**。

实测对比：同一种 FSR，直连 ADC（PA1）vs 经运放缓冲（PA2），运放侧噪声标准差明显更低、长时间采样漂移更小。

## 软件模块

```
Core/
├── Inc/
│   ├── main.h
│   ├── gpio.h / adc.h / i2c.h / usart.h    ← CubeMX 生成
│   ├── oled.h                                ← 复用项目一
│   ├── debug.h                               ← 串口 printf
│   ├── fsr_adc.h                             ← ADC 采集 + 滤波
│   └── calib.h                               ← 传感器标定
└── Src/
    ├── main.c
    ├── gpio.c / adc.c / i2c.c / usart.c     ← CubeMX 生成
    ├── oled.c                                ← 复用项目一
    ├── debug.c                               ← fputc 重定向
    ├── fsr_adc.c                             ← HAL_ADC_GetValue, 滑动平均, DMA
    └── calib.c                               ← 分段线性化, ADC值→力值(g)
```

## CubeMX 配置

| 外设 | 配置 | 用途 |
|---|---|---|
| HSE + PLL | 8MHz ×9 → 72MHz | 系统时钟 |
| ADC1 | 12bit, IN1(PA1) + IN2(PA2) | FSR 双通道采集 |
| I2C1 | 100kHz, PB6(SCL) / PB7(SDA) | OLED 显示 |
| USART1 | 115200, PA9(TX) / PA10(RX) | 调试 + 数据上传 |
| GPIO | PC13 Output | LED 心跳 |

## 标定方法

用已知重量（硬币、手机）做 4~5 个力值参考点，每点采 16 次取平均。FSR 呈非线性特性（低压段斜率陡、高压段平缓），采用分段线性化：`calib.c` 中维护一个查找表，相邻两点间线性插值，将 ADC 原始值换算为近似力值（克）。

## 开发与构建

1. CubeMX 打开 `fsr_signal_chain.ioc` → GENERATE CODE
2. Keil5 打开 `MDK-ARM/` 下工程 → Build
3. ST-Link 烧录 → PC13 LED 闪烁 → OLED 显示压力值
4. 串口工具（SSCOM, 115200）查看原始 ADC 数据
